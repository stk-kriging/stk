DESCRIPTION 

 Example 03 constructs a kriging approximation in 2D
 ==================================================
    Construct a kriging approximation in 2D. In this example, the model is
    estimated from data.

CROSS-REFERENCE INFORMATION

This function calls:
  • stk_predict STK_PREDICT performs a kriging prediction from data
  • stk_param_estim STK_PARAM_ESTIM estimates the parameters of a covariance function.
  • stk_sampling_maximinlhs STK_SAMPLING_MAXIMINLHS generates a "maximin" LHS design.
  • stk_feval STK_FEVAL evaluates a function at given evaluation points.
  • stk_model STK_MODEL generates a model with default covariance parameters.
