CROSS-REFERENCE INFORMATION

This function calls:
  • stk_predict STK_PREDICT performs a kriging prediction from data
  • stk_plot1dsim STK_PLOT1DSIM is a plot helper for using with conditional sample paths
  • stk_sampling_regulargrid STK_SAMPLING_REGULARGRID builds a regular grid.
  • stk_conditioning STK_CONDITIONING produces conditioned sample paths.
  • stk_feval STK_FEVAL evaluates a function at given evaluation points.
  • stk_generate_samplepaths STK_GENERATE_SAMPLEPATHS generates sample paths of a Gaussian process.
  • stk_model STK_MODEL generates a model with default covariance parameters.

0041 %% Define a 1d test function
0042 
0043 f = @(x)( -(0.7*x+sin(5*x+1)+0.1*sin(10*x)) );  % define a 1D test function
0044 DIM = 1;                                        % dimension of the factor space
0045 box = [-1.0; 1.0];                              % factor space
0046 
0047 NT = 400; % nb of points in the grid
0048 xt = stk_sampling_regulargrid( NT, DIM, box );
0049 zt = stk_feval( f, xt );
0050 
0051 
0052 %% Generate observations
0053 %
0054 % The objective is to construct an approximation of f and to simulate
0055 % conditioned sample paths from NI observations. The observation locations
0056 % are chosen as a subset of xt.a.
0057 %
0058 
0059 NI = 6;                               % nb of evaluations that will be used
0060 xi_ind  = [1 20 90 200 300 350];      %
0061 xi.a = xt.a(xi_ind, 1);
0062 zi = stk_feval( f, xi );              % evaluation results
0063 
0064 
0065 %% Specification of the model
0066 %
0067 % We choose a Matern covariance with "fixed parameters" (in other
0068 % words, the parameters of the covariance function are provided by the user
0069 % rather than estimated from data).
0070 %
0071 
0072 % The following line defines a model with a constant but unknown mean
0073 % (ordinary kriging) and a Matern covariance function. (Some default
0074 % parameters are also set, but we override them below.)
0075 model = stk_model('stk_materncov_iso');
0076 
0077 % Parameters for the Matern covariance
0078 % ("help stk_materncov_iso" for more information)
0079 SIGMA2 = 1.0;  % variance parameter
0080 NU     = 4.0;  % regularity parameter
0081 RHO1   = 0.4;  % scale (range) parameter
0082 model.param = log([SIGMA2; NU; 1/RHO1]);
0083 
0084 
0085 %% Carry out the kriging prediction and generate conditional sample paths
0086 %
0087 
0088 % Carry out the kriging prediction at points xt.a
0089 [zp, lambda] = stk_predict( model, xi, zi, xt );
0090 
0091 % Generate (unconditional) sample paths according to the model
0092 NB_PATHS = 10;
0093 zsim = stk_generate_samplepaths( model, xt, NB_PATHS );
0094 
0095 % Condition sample paths on the observations
0096 zsimc = stk_conditioning( lambda, zi, zsim, xi_ind );
0097 
0098 % Display the result
0099 stk_plot1dsim ( xi, zi, xt, zt, zp, zsimc );
