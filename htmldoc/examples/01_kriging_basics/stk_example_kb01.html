stk_example_kb01

PURPOSE

STK_EXAMPLE_KB01 constructs a kriging approximation in 1D.

SYNOPSIS

This is a script file.

DESCRIPTION 

 STK_EXAMPLE_KB01 constructs a kriging approximation in 1D.

   This example shows how to construct an ordinary kriging predictor for a
   scalar input. The covariance function is assumed to be fully known (i.e., no
   parameter estimation is performed here).

0038 %% Define a 1d test function
0039 
0040 f = @(x)( -(0.7*x+sin(5*x+1)+0.1*sin(10*x)) );  % define a 1D test function
0041 DIM = 1;                                        % dimension of the factor space
0042 BOX = [-1.0; 1.0];                              % factor space
0043 
0044 NT = 400; % nb of points in the grid
0045 xt = stk_sampling_regulargrid(NT, DIM, BOX);
0046 zt = stk_feval(f, xt);
0047 
0048 figure(1);
0049 set(gcf, 'Name', 'Plot of the function to be approximated');
0050 plot(xt, zt); xlabel('x'); ylabel('z');
0051 
0052 
0053 %% Generate a space-filling design
0054 %
0055 % The objective is to construct an approximation of f with a budget of NI
0056 % evaluations performed on a "space-filling design".
0057 %
0058 % A regular grid (i.e., a grid with constant spacing) is constructed using
0059 % stk_sampling_regulargrid(), which is equivalent to linspace() in this
0060 % 1d example.
0061 %
0062 
0063 NI = 6;                                        % nb of evaluations that will be used
0064 xi = stk_sampling_regulargrid(NI, DIM, BOX);   % evaluation points
0065 zi = stk_feval(f, xi);                         % evaluation results
0066 
0067 
0068 %% Specification of the model
0069 %
0070 % We choose a Matern covariance with "fixed parameters" (in other
0071 % words, the parameters of the covariance function are provided by the user
0072 % rather than estimated from data).
0073 %
0074 
0075 % The following line defines a model with a constant but unknown mean
0076 % (ordinary kriging) and a Matern covariance function. (Some default
0077 % parameters are also set, but we override them below.)
0078 model = stk_model('stk_materncov_iso');
0079 
0080 % NOTE: the suffix '_iso' indicates an ISOTROPIC covariance function, but the
0081 % distinction isotropic / anisotropic is irrelevant here since DIM = 1.
0082 
0083 % Parameters for the Matern covariance function
0084 % ("help stk_materncov_iso" for more information)
0085 SIGMA2 = 1.0;  % variance parameter
0086 NU     = 4.0;  % regularity parameter
0087 RHO1   = 0.4;  % scale (range) parameter
0088 model.param = log([SIGMA2; NU; 1/RHO1]);
0089 
0090 
0091 %% Carry out the kriging prediction and display the result
0092 %
0093 % The result of a kriging predicition is provided by stk_predict() in an object
0094 % zp of type stk_dataframe, with two columns: "zp.mean" (the kriging mean) and
0095 % "zp.var" (the kriging variance).
0096 %
0097 
0098 % Carry out the kriging prediction at points xt
0099 zp = stk_predict(model, xi, zi, xt);
0100 
0101 % Display the result
0102 stk_plot1d(xi, zi, xt, zt, zp);
0103 t = 'Kriging prediction based on noiseless observations';
0104 set(gcf, 'Name', t); title(t);
0105 xlabel('x'); ylabel('z');

0109 
0110 NOISEVARIANCE = (1e-1)^2;
0111 
0112 % Now the observations are perturbed by an additive Gaussian noise
0113 noise = sqrt(NOISEVARIANCE) * randn(size(zi));
0114 zi_n = zi + noise;
0115 
0116 % We also include the observation noise in the model
0117 model_n = model;
0118 model_n.lognoisevariance = log(NOISEVARIANCE);
0119 
0120 % Carry out the kriging prediction at locations xt
0121 zp_n = stk_predict(model_n, xi, zi_n, xt);
