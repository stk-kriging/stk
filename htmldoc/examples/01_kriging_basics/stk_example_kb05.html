SYNOPSIS

DESCRIPTION 

 STK_EXAMPLE_KB05  Generation of conditioned sample paths

 A Matern Gaussian process model is used, with constant but unknown mean
 (ordinary kriging) and known covariance parameters.

 Given noiseless observations from the unknown function, a batch of conditioned
 sample paths is drawn using the "conditioning by kriging" technique. In short,
 this means that unconditioned sample path are simulated first (using
 stk_generate_samplepaths), and then conditioned on the observations by kriging
 (using stk_conditioning).

0042 %% Define a 1d test function
0043 
0044 f = @(x)(- (0.7 * x + sin (5 * x + 1) + 0.1 * sin (10 * x)));
0045 DIM = 1;            % dimension of the factor space
0046 BOX = [-1.0; 1.0];  % factor space
0047 
0048 NT = 400;  % nb of points in the grid
0049 xt = stk_sampling_regulargrid (NT, DIM, BOX);
0050 zt = stk_feval (f, xt);
0051 
0052 
0053 %% Generate observations
0054 %
0055 % The objective is to construct an approximation of f and to simulate
0056 % conditioned sample paths from NI observations. The observation locations
0057 % are chosen as a subset of xt.
0058 %
0059 
0060 xi_ind = [1 20 90 200 300 350];  % indices of evaluation points in xt
0061 xi = xt(xi_ind, 1);              % evaluation points
0062 zi = stk_feval (f, xi);          % evaluation results
0063 
0064 
0065 %% Specification of the model
0066 %
0067 % We choose a Matern covariance with "fixed parameters" (in other
0068 % words, the parameters of the covariance function are provided by the user
0069 % rather than estimated from data).
0070 %
0071 
0072 % The following line defines a model with a constant but unknown mean
0073 % (ordinary kriging) and a Matern covariance function. (Some default
0074 % parameters are also set, but we override them below.)
0075 model = stk_model ('stk_materncov_iso');
0076 
0077 % Parameters for the Matern covariance
0078 % ("help stk_materncov_iso" for more information)
0079 SIGMA2 = 1.0;  % variance parameter
0080 NU     = 4.0;  % regularity parameter
0081 RHO1   = 0.4;  % scale (range) parameter
0082 model.param = log ([SIGMA2; NU; 1/RHO1]);

0089 zsim = stk_generate_samplepaths (model, xt, NB_PATHS);
