 Example 02 constructs a kriging approximation in 1D
 ===================================================
    Construct a kriging approximation in 1D. In this example, the model is
    estimated from data.

CROSS-REFERENCE INFORMATION

This function calls:
  • stk_predict STK_PREDICT performs a kriging prediction from data
  • stk_plot1d STK_PLOT1D is a convenient plot function in 1D
  • stk_param_estim STK_PARAM_ESTIM estimates the parameters of a covariance function.
  • stk_sampling_randunif STK_SAMPLING_RANDUNIF generates uniformly distributed points.
  • stk_sampling_regulargrid STK_SAMPLING_REGULARGRID builds a regular grid.
  • stk_feval STK_FEVAL evaluates a function at given evaluation points.
  • stk_model STK_MODEL generates a model with default covariance parameters.

0042 %% Define a 1d test function (the same as in example01.m)
0043 
0044 f = @(x)( -(0.8*x+sin(5*x+1)+0.1*sin(10*x)) );  % define a 1D test function
0045 DIM = 1;                                        % dimension of the factor space
0046 box = [-1.0; 1.0];                              % factor space
0047 
0048 NT = 400; % nb of points in the grid
0049 xt = stk_sampling_regulargrid( NT, DIM, box );
0050 zt = stk_feval( f, xt );
0051 
0052 
0053 %% Generate a random sampling plan
0054 %
0055 % The objective is to construct an approximation of f with a budget of NI
0056 % evaluations performed on a randomly generated (uniform) design.
0057 %
0058 % Change the value of NOISEVARIANCE to add a Gaussian evaluation noise on
0059 % the observations.
0060 %
0061 
0062 NOISEVARIANCE = 0.0;
0063 
0064 NI = 6;                                     % nb of evaluations that will be used
0065 xi = stk_sampling_randunif(NI, DIM, box);   % evaluation points
0066 zi = stk_feval(f, xi);                      % evaluation results
0067 
0068 if NOISEVARIANCE > 0,
0069     zi.a = zi.a + sqrt(NOISEVARIANCE) * randn(NI,1);
0070     % (don't forget that the data is in the ".a" field!)
0071 end
0072 
0073 
0074 %% Specification of the model
0075 %
0076 % We choose a Matern covariance, the parameters of which will be estimated from the data.
0077 %
0078 % The values of the parameters that are provided here, including the noise variance, are
0079 % only used as an initial point for the optimization algorithm used in stk_param_estim().
0080 %
0081 
0082 % The following line defines a model with a constant but unknown mean (ordinary
0083 % kriging) and a Matern covariance function. (Some default parameters are also
0084 % set, but they will be replaced below by estimated parameters.)
0085 model = stk_model('stk_materncov_iso');
0086 
0087 % Noise variance
0088 if NOISEVARIANCE > 0,
0089     model.lognoisevariance = log( NOISEVARIANCE );
0090 else
0091     % Even if we don't assume that the observations are noisy,
0092     % it is wiser to add a small "regularization noise".
0093     model.lognoisevariance = log( 100 * eps );
0094 end
0095 
0096 
0097 %% Estimatation the parameters of the covariance
0098 %
0099 % Here, the parameters of the Matern covariance function are estimated by the
0100 % REML (REstricted Maximum Likelihood) method using stk_param_estim().
0101 %
0102 
0103 % Initial guess for the parameters for the Matern covariance
0104 % (see "help stk_materncov_iso" for more information)
0105 SIGMA2 = 1.0;  % variance parameter
0106 NU     = 4.0;  % regularity parameter
0107 RHO1   = 0.4;  % scale (range) parameter
0108 param0 = log([SIGMA2; NU; 1/RHO1]);
0109 
0110 model.param = stk_param_estim(model, xi, zi, param0);
0111 
0112 
0113 %% carry out kriging prediction
0114 
0115 zp = stk_predict(model, xi, zi, xt);
0116 
0117

0119 
0120 stk_plot1d(xi,zi,xt,zt,zp)
