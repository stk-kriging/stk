 Example 02 constructs a kriging approximation in 1D
 ===================================================
    Construct a kriging approximation in 1D. In this example, the model is
    estimated from data.

CROSS-REFERENCE INFORMATION

This function calls:
  • stk_param_estim STK_PARAM_ESTIM estimates the parameters of the covariance from data
  • stk_predict STK_PREDICT performs a kriging prediction from data
  • stk_sampling_cartesiangrid STK_SAMPLING_CARTESIANGRID builds a regular grid
  • stk_sampling_randunif STK_SAMPLING_RANDUNIF yields uniformly distributed points on a box domain
  • stk_feval STK_FEVAL evaluates a function at given locations points
  • stk_plot1d STK_PLOT1D is a convenient plot function in 1D

0037 %% Define a 1d test function (the same as in example01.m)
0038 
0039 f = @(x)( -(0.8*x+sin(5*x+1)+0.1*sin(10*x)) );  % define a 1D test function
0040 DIM = 1;                                        % dimension of the factor space
0041 box = [-1.0; 1.0];                              % factor space
0042 
0043 NT = 400; % nb of points in the grid
0044 xt = stk_sampling_cartesiangrid( NT, DIM, box );
0045 zt = stk_feval( f, xt );
0046 
0047 %% Generate a space-filling design
0048 %
0049 % The objective is to construct an approximation of f with a budget of NI
0050 % evaluations performed on a "space-filling design".
0051 %
0052 % A regular grid (i.e., a grid with constant spacing) is constructed using
0053 % stk_sampling_cartesiangrid(), which is equivalent to linspace() for this
0054 % simple 1d example.
0055 %
0056 % Change the value of NOISEVARIANCE to add a Gaussian evaluation noise on
0057 % the observations.
0058 %
0059 
0060 NOISEVARIANCE = 0;
0061 
0062 NI = 6;                                         % nb of evaluations that will be used
0063 xi = stk_sampling_randunif( NI, DIM, box); % evaluation points
0064 zi = stk_feval( f, xi );                        % evaluation results
0065 
0066 if NOISEVARIANCE > 0,
0067     zi.a = zi.a + sqrt(NOISEVARIANCE) * randn(NI,1);
0068     % (don't forget that the data is in the ".a" field!)
0069 end
0070 
0071 
0072 %% Specification of the model
0073 %
0074 % We choose a Matern covariance, the parameters of which will be estimated from the data.
0075 %
0076 % The values of the parameters that are provided here, including the noise variance, are
0077 % only used as an initial point for the optimization algorithm used in stk_param_estim().
0078 %
0079 
0080 % Parameters for the Matern covariance (see "help stk_materncov_iso" for more information)
0081 SIGMA2 = 1.0;  % variance parameter
0082 NU     = 4.0;  % regularity parameter
0083 RHO1   = 0.4;  % scale (range) parameter
0084 
0085 % Specification of the model (see "help stk_model" for more information)
0086 model.covariance_type  = 'stk_materncov_iso';     % isotropic Matern covariance function
0087 model.order = 0;                                  % ordinary kriging (i.e., constant mean)
0088 model.param = [log(SIGMA2),log(NU),log(1/RHO1)]'; % vector of parameters
0089 
0090 % Noise variance
0091 if NOISEVARIANCE > 0,
0092     model.lognoisevariance = log( 3 * NOISEVARIANCE );
0093     % (this is not the true value of the noise variance !)
0094 else
0095     % Even if we don't assume that the observations are noisy,
0096     % it is wiser to add a small "regularization noise".
0097     model.lognoisevariance = log( 100 * eps );
0098 end
0099 
0100 
0101 %% Estimatation the parameters of the covariance
0102 %
0103 % The parameters are estimated by the REML (REstricted Maximum Likelihood) method.
0104 %
0105 
0106 model.param = stk_param_estim( model.param, xi, zi, model);
0107 
0108 %% carry out kriging prediction
0109 
0110 zp = stk_predict(xi, zi, xt, model);

0112 %% display results
0113 
0114 stk_plot1d(xi,zi,xt,zt,zp)
