 Example 02 constructs a kriging approximation in 1D

    estimated from data.

  • stk_sampling_randunif STK_SAMPLING_RANDUNIF yields uniformly distributed points on a box domain

0037 %% Define a 1d test function (the same as in example01.m)
0038 
0039 f = @(x)( -(0.8*x+sin(5*x+1)+0.1*sin(10*x)) );  % define a 1D test function
0040 DIM = 1;                                        % dimension of the factor space
0041 box = [-1.0; 1.0];                              % factor space
0042 
0043 NT = 400; % nb of points in the grid
0044 xt = stk_sampling_cartesiangrid( NT, DIM, box );
0045 zt = stk_feval( f, xt );
0046 
0047 %% Generate a random sampling plan
0048 %
0049 % The objective is to construct an approximation of f with a budget of NI
0050 % evaluations performed on a randomly generated (uniform) design.
0051 %
0052 % Change the value of NOISEVARIANCE to add a Gaussian evaluation noise on
0053 % the observations.
0054 %
0055 
0056 NOISEVARIANCE = 0;
0057 
0058 NI = 6;                                     % nb of evaluations that will be used
0059 xi = stk_sampling_randunif(NI, DIM, box);   % evaluation points
0060 zi = stk_feval(f, xi);                      % evaluation results
0061 
0062 if NOISEVARIANCE > 0,
0063     zi.a = zi.a + sqrt(NOISEVARIANCE) * randn(NI,1);
0064     % (don't forget that the data is in the ".a" field!)
0065 end
0066 
0067 
0068 %% Specification of the model
0069 %
0070 % We choose a Matern covariance, the parameters of which will be estimated from the data.
0071 %
0072 % The values of the parameters that are provided here, including the noise variance, are
0073 % only used as an initial point for the optimization algorithm used in stk_param_estim().
0074 %
0075 
0076 % Parameters for the Matern covariance (see "help stk_materncov_iso" for more information)
0077 SIGMA2 = 1.0;  % variance parameter
0078 NU     = 4.0;  % regularity parameter
0079 RHO1   = 0.4;  % scale (range) parameter
0080 
0081 % Specification of the model (see "help stk_model" for more information)
0082 model.covariance_type  = 'stk_materncov_iso';     % isotropic Matern covariance function
0083 model.order = 0;                                  % ordinary kriging (i.e., constant mean)
0084 model.param = [log(SIGMA2),log(NU),log(1/RHO1)]'; % vector of parameters
0085 
0086 % Noise variance
0087 if NOISEVARIANCE > 0,
0088     model.lognoisevariance = log( 3 * NOISEVARIANCE );
0089     % (this is not the true value of the noise variance !)
0090 else
0091     % Even if we don't assume that the observations are noisy,
0092     % it is wiser to add a small "regularization noise".
0093     model.lognoisevariance = log( 100 * eps );
0094 end
0095 
0096 %% Estimatation the parameters of the covariance
0097 %
0098 % The parameters are estimated by the REML (REstricted Maximum Likelihood) method.
0099 %
0100 
0101 model.param = stk_param_estim( model.param, xi, zi, model);

0104
