DESCRIPTION 

 Example 01 constructs a kriging approximation in 1D
 ===================================================
    Construct a kriging approximation in 1D. In this example, the model is
    fixed by the user.

CROSS-REFERENCE INFORMATION

This function calls:
  • stk_predict STK_PREDICT performs a kriging prediction from data
  • stk_plot1d STK_PLOT1D is a convenient plot function in 1D
  • stk_sampling_regulargrid STK_SAMPLING_REGULARGRID builds a regular grid.
  • stk_feval STK_FEVAL evaluates a function at given evaluation points.
  • stk_model STK_MODEL generates a model with default covariance parameters.

0042 %% Define a 1d test function
0043 
0044 f = @(x)( -(0.7*x+sin(5*x+1)+0.1*sin(10*x)) );  % define a 1D test function
0045 DIM = 1;                                        % dimension of the factor space
0046 box = [-1.0; 1.0];                              % factor space
0047 
0048 NT = 400; % nb of points in the grid
0049 xt = stk_sampling_regulargrid( NT, DIM, box );
0050 zt = stk_feval( f, xt );
0051 
0052 figure(1); set( gcf, 'Name', 'Plot of the function to be approximated');
0053 plot( xt.a, zt.a ); xlabel('x'); ylabel('z');
0054 
0055 % In STK, the inputs and outputs are members of a structure array
0056 % The field 'a' is used to store the numerical values
0057 
0058 
0059 %% Generate a space-filling design
0060 %
0061 % The objective is to construct an approximation of f with a budget of NI
0062 % evaluations performed on a "space-filling design".
0063 %
0064 % A regular grid (i.e., a grid with constant spacing) is constructed using
0065 % stk_sampling_regulargrid(), which is equivalent to linspace() in this
0066 % 1d example.
0067 %
0068 
0069 NI = 6;                                         % nb of evaluations that will be used
0070 xi = stk_sampling_regulargrid( NI, DIM, box); % evaluation points
0071 zi = stk_feval( f, xi );                        % evaluation results
0072 
0073 
0074 %% Specification of the model
0075 %
0076 % We choose a Matern covariance with "fixed parameters" (in other
0077 % words, the parameters of the covariance function are provided by the user
0078 % rather than estimated from data).
0079 %
0080 
0081 % The following line defines a model with a constant but unknown mean
0082 % (ordinary kriging) and a Matern covariance function. (Some default
0083 % parameters are also set, but we override them below.)
0084 model = stk_model('stk_materncov_iso');
0085 
0086 % NOTE: the suffix '_iso' indicates an ISOTROPIC covariance function, but the
0087 % distinction isotropic / anisotropic is irrelevant here since DIM = 1.
0088 
0089 % Parameters for the Matern covariance function
0090 % ("help stk_materncov_iso" for more information)
0091 SIGMA2 = 1.0;  % variance parameter
0092 NU     = 4.0;  % regularity parameter
0093 RHO1   = 0.4;  % scale (range) parameter
0094 model.param = log([SIGMA2; NU; 1/RHO1]);
0095 
0096 
0097 %% Carry out the kriging prediction and display the result
0098 %
0099 % The result of a kriging predicition is provided by stk_predict() in a
0100 % structure, called "zp" in this example, which has two fields: "zp.a" (the
0101 % kriging mean) and "zp.v" (the kriging variance).
0102 %
0103 
0104 % Carry out the kriging prediction at points xt.a
0105 zp = stk_predict( model, xi, zi, xt );
0106 
0107 % Display the result
0108 stk_plot1d( xi, zi, xt, zt, zp );
0109 t = 'Kriging prediction based on noiseless observations';
0110 set( gcf, 'Name', t ); title(t);
0111 xlabel('x'); ylabel('z');
0112 
0113 
0114 %% Repeat the experiment in a noisy setting
0115 
0116 NOISEVARIANCE = (1e-1)^2;
0117 
0118 % Now the observations are perturbed by an additive Gaussian noise
0119 noise = sqrt(NOISEVARIANCE) * randn(size(zi.a));
0120 zi_n = struct( 'a', zi.a + noise  );
0121 
0122 % We also include the observation noise in the model
0123 model_n = model;
0124 model_n.lognoisevariance = log(NOISEVARIANCE);
0125 
0126 % Carry out the kriging prediction at locations xt.a
0127 zp_n = stk_predict( model_n, xi, zi_n, xt );
0128 
0129 % Display the result
0130 stk_plot1d( xi, zi_n, xt, zt, zp_n );
0131 t = 'Kriging prediction based on noisy observations';
0132 set( gcf, 'Name', t ); title(t);
