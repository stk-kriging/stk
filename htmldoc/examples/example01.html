 Example 01 constructs a kriging approximation in 1D
 ===================================================
    Construct a kriging approximation in 1D. In this example, the model is
    fixed by the user.

CROSS-REFERENCE INFORMATION

This function calls:
  • stk_predict STK_PREDICT performs a kriging prediction from data
  • stk_sampling_cartesiangrid STK_SAMPLING_CARTESIANGRID builds a regular grid
  • stk_feval STK_FEVAL evaluates a function at given locations points
  • stk_plot1d STK_PLOT1D is a convenient plot function in 1D

0037 f = @(x)( -(0.7*x+sin(5*x+1)+0.1*sin(10*x)) );  % define a 1D test function
0038 DIM = 1;                                        % dimension of the factor space
0039 box = [-1.0; 1.0];                              % factor space

0043 zt = stk_feval( f, xt );

0052 %% Generate a space-filling design
0053 %
0054 % The objective is to construct an approximation of f with a budget of NI
0055 % evaluations performed on a "space-filling design".
0056 %
0057 % A regular grid (i.e., a grid with constant spacing) is constructed using
0058 % stk_sampling_cartesiangrid(), which is equivalent to linspace() in this
0059 % 1d example.
0060 %
0061 
0062 NI = 6;                                         % nb of evaluations that will be used
0063 xi = stk_sampling_cartesiangrid( NI, DIM, box); % evaluation points
0064 zi = stk_feval( f, xi );                        % evaluation results
0065

0067 %% Specification of the model
0068 %
0069 % We choose a Matern covariance with "fixed parameters" (in other
0070 % words, the parameters of the covariance function are provided by the user
0071 % rather than estimated from data).
0072 %
0073 
0074 % Parameters for the Matern covariance ("help stk_materncov_iso" for more information)
0075 SIGMA2 = 1.0;  % variance parameter
0076 NU     = 4.0;  % regularity parameter
0077 RHO1   = 0.4;  % scale (range) parameter
0078 
0079 % Specification of the model (see "help stk_model" for more information)
0080 model.covariance_type  = 'stk_materncov_iso';     % isotropic Matern covariance function
0081 model.order = 0;                                  % ordinary kriging (i.e., constant mean)
0082 model.param = [log(SIGMA2),log(NU),log(1/RHO1)]'; % vector of parameters
0083 
0084 
0085 %% Carry out the kriging prediction and display the result
0086 %
0087 % The result of a kriging predicition is provided by stk_predict() in a
0088 % structure, called "zp" in this example, which has two fields: "zp.a" (the
0089 % kriging mean) and "zp.v" (the kriging variance).
0090 %
0091 
0092 % Carry out the kriging prediction at points xt.a
0093 zp = stk_predict( xi, zi, xt, model );
