stk_param_relik

PURPOSE

STK_REMLQRG computes the restricted likelihood of a model given data.

SYNOPSIS

function [rl, drl_param, drl_lnv] = stk_param_relik (model, xi, yi)

DESCRIPTION 

 STK_REMLQRG computes the restricted likelihood of a model given data.

 CALL: [ARL, dARL_dtheta, dARL_dLNV] = stk_param_relik (MODEL, XI, YI)

   computes the opposite of the restricted likelihood (denoted by ARL for
   Anti-Restricted Likelihood) of MODEL given the data (XI, YI). The function
   also returns the gradient dARL_dtheta of ARL with respect to the parameters
   of the covariance function and the derivative dARL_dLNV of ARL with respect
   to the logarithm of the noise variance.

 EXAMPLE: see paramestim/stk_param_estim.m

0047 % an object, this is actually a call to subsasgn() in disguise).
0048 param = model.param(:);

0050 PARAMPRIOR = isfield(model, 'prior');
0051 NOISYOBS   = isfield(model, 'lognoisevariance');
0052 NOISEPRIOR = isfield(model, 'noiseprior');
0053 
0054 if ~NOISYOBS,
0055     if NOISEPRIOR,
0056         error(['Having a prior on the noise variance when there is' ...
0057             'no observation noise doesn''t make sense...']);
0058     else
0059         % log(eps) is harmless
0060         model.lognoisevariance = log(eps);
0061     end

0067 
0068 [K, P] = stk_make_matcov(model, xi);
0069 q = size(P, 2);
0070 
0071 [Q, R_ignored] = qr(P); %#ok<NASGU> %the second argument *must* be here
0072 W = Q(:, (q+1):n);
0073 Wyi = W' * double(yi);

0075 G = W' * (K * W);

0077 Ginv = inv(G);
0078 WKWinv_Wyi = Ginv * Wyi; %#ok<MINV>

0095 else
0096     noiseprior = 0;
0097 end

0099 rl = 1/2 * ((n-q) * log(2*pi) + ldetWKW + attache + prior + noiseprior);

0112     end
