stk_isequal_tolrel

PURPOSE

STK_ISEQUAL_TOLREL tests approximate equality of two matrices or structures.

SYNOPSIS

function res = stk_isequal_tolrel(a, b, tolrel)

DESCRIPTION 

 STK_ISEQUAL_TOLREL tests approximate equality of two matrices or structures.

 CALL: BOOL = stk_isequal_tolrel(A, B, TOLREL)

   returns true if A and B are numeric arrays of the same size, such that for
   any pair (a, b) of corresponding entries,

      abs(b - a) <= TOLABS,                                                (1)

   where

      TOLABS = max(max(A(:)), max(B(:))) * TOLREL.

   For numeric array, the function returns false is either

    * the array don't have identical sizes, or
    * the array have identical sizes but (1) doesn't hold.

   If A and B are structures with the same list of fieldnames, the function
   works recursively on the fields, and returns true iff all the fields are
   approximately equal.

 CALL: b = stk_isequal_tolrel(a, b)

   uses the default value 1e-8 for TOLREL.

 See also isequal.

0056 function res = stk_isequal_tolrel(a, b, tolrel)
0057 
0058 DEFAULT_TOLREL = 1e-8;
0059 
0060 stk_narginchk(2, 3);
0061 
0062 if nargin == 2,
0063     tolrel = DEFAULT_TOLREL;
0064 end
0065 
0066 if isstruct(a) && isstruct(b),
0067     
0068     L = fieldnames(a);
0069     if ~isequal(fieldnames(b), L),
0070         res = false;
0071         return;
0072     end
0073     res = true;
0074     for k = 1:length(L),
0075         if ~isfield(b, L{k}),
0076             res = false;
0077             return;
0078         end
0079         res = stk_isequal_tolrel(a.(L{k}), b.(L{k}), tolrel);
0080         if ~ res, return; end
0081     end
0082     
0083 elseif isnumeric(a) && isnumeric(b)
0084     
0085     aa = a(:);
0086     bb = b(:);
0087     tolabs = max(abs(aa), abs(bb)) * tolrel;
0088     res = any(abs(bb - aa) <= tolabs);
0089     
0090 else
0091     
0092     res = false;
0093     
0094 end
