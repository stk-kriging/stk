stk_options_get

PURPOSE

STK_OPTIONS_GET returns the value of one or all STK options.

SYNOPSIS

function argout = stk_options_get(varargin)

DESCRIPTION 

 STK_OPTIONS_GET returns the value of one or all STK options.

CROSS-REFERENCE INFORMATION

0029 function argout = stk_options_get(varargin)
0030 
0031 opts = stk_options_set();
0032 
0033 switch nargin
0034     
0035     case 0, % nothing to do, just return the output
0036         argout = opts;
0037         
0038     case 1,
0039         argout = opts.(varargin{1});
0040         
0041     case 2,
0042         argout = opts.(varargin{1}).(varargin{2});

0047 end
