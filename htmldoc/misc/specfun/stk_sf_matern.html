stk_sf_matern

PURPOSE

STK_SF_MATERN computes the Matern correlation function.

SYNOPSIS

function k = stk_sf_matern (Nu, h, diff)

DESCRIPTION 

 STK_SF_MATERN computes the Matern correlation function.

 CALL: K = stk_sf_matern (NU, H)

    computes the value of the Matern correlation function of order NU at
    distance H. Note that the Matern correlation function is a valid
    correlation function for all dimensions.

 CALL: K = stk_sf_matern (NU, H, DIFF)

    computes the derivative of the Matern correlation function of order NU, at
    distance H, with respect to the order NU if DIFF is equal to 1, or with
    respect the distance H if DIFF is equal to 2. (If DIFF is equal to -1,
    this is the same as K = stk_sf_matern(NU, H).)

 See also: stk_sf_matern32, stk_sf_matern52

0045 function k = stk_sf_matern (Nu, h, diff)
0046 
0047 if nargin > 3,
0048    stk_error ('Too many input arguments.', 'TooManyInputArgs');
0049 end
0050 
0051 % default: compute the value (not a derivative)
0052 if nargin < 3,
0053     diff = -1;
0054 end
0055 
0056 [N, M] = size (h);
0057 hp = abs (reshape (h, N * M, 1));
0058 t = 2 * sqrt (Nu) * hp;
0059 z = 2 ^ (Nu - 1) * gamma (Nu) * t .^ (-Nu);
0060 I = ~ isinf (z);
0061 
0062 if diff == -1
0063     
0064     k = ones (N * M, 1);
0065     k(I) = 1 ./ z(I) .* besselk_ (Nu, t(I));
0066     
0067 elseif diff == 1  % numerical derivative wrt Nu
0068     
0069     itermax = 2;
0070     delta = 1e-4;
0071     dk = zeros (N * M, itermax);
0072     for l= 1:itermax
0073         Nu_p = Nu + 2 ^ (l - 1) * delta;
0074         Nu_m = Nu - 2 ^ (l - 1) * delta;
0075         t_p = 2 * sqrt (Nu_p) * hp;
0076         t_m = 2 * sqrt (Nu_m) * hp;
0077         k_p = 1 / (2 ^ (Nu_p - 1) * gamma (Nu_p)) .* t_p(I) .^ Nu_p .* ...
0078             besselk_ (Nu_p, t_p(I));
0079         k_m = 1 / (2 ^ (Nu_m - 1) * gamma (Nu_m)) .* t_m(I) .^ Nu_m .* ...
0080             besselk_ (Nu_m, t_m(I));
0081         dk(I, l) =  k_p - k_m;
0082     end
0083     k = 1 / (12 * delta)* (- dk(:, 2) + 8 * dk(:, 1));
0084     
0085 elseif diff == 2  % deriv. wrt h
0086     
0087     k = zeros (N * M, 1);
0088     dtdh = 2 * sqrt (Nu);
0089     k(I)  = - dtdh ./ z(I) .* besselk_ (Nu - 1, t(I));
0090     
0091 end
0092 
0093 k = reshape (k, N, M);
0094 
0095 end % function stk_sf_matern
