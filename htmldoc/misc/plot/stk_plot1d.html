stk_plot1d

PURPOSE

STK_PLOT1D is a convenient plot function in 1D

SYNOPSIS

function stk_plot1d (xi, zi, xt, zt, zp, zsim)

DESCRIPTION 

 STK_PLOT1D is a convenient plot function in 1D

 CALL: stk_plot1d (xi, zi, xt, zt, zp)

 STK_PLOT1D plots the result of a 1D approximation
