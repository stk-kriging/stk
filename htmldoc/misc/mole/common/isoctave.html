isoctave

PURPOSE

ISOCTAVE returns true if Octave is used as an interpreter, false otherwise

SYNOPSIS

function octave_in_use = isoctave

DESCRIPTION 

 ISOCTAVE returns true if Octave is used as an interpreter, false otherwise
