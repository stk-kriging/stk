index

PURPOSE

INDEX finds the first (or last) occurence of one string within another.

SYNOPSIS

function n = index (s, t, direction)

DESCRIPTION 

 INDEX finds the first (or last) occurence of one string within another.

 CALL: index(S, T)
 CALL: index(S, T, "first")
   Returns the position of the first occurrence of the string T in the
   string S, or 0 if no occurrence is found. S may also be a cell array
   of strings.

 CALL: index(S, T, "last")
   Returns the position of the last occurrence.

 EXAMPLES:
   index("Teststring", "t")            returns 4
   index('Teststring', 't', 'last')    returns 6
   index({'Hello', 'Toto'}, 'o')        returns [5 2]

 SEE ALSO:
   strfind

0056 function n = index (s, t, direction)
0057 
0058 if nargin < 3, direction = 'first'; end
0059 
0060 if ischar(s)
0061     if size(s, 1) > 1
0062         s = cellstr (s);  % Handle string arrays by conversion to cellstr
0063     end
0064 elseif ~iscellstr(s)
0065     error ('index: S must be a string, string array, or cellstr');
0066 end
0067 
0068 f = strfind (s, t);
0069 if (isempty (f))
0070     f = 0;
0071 elseif (iscell (f))
0072     f(cellfun ('isempty', f)) = {0};
0073 end
0074 
0075 direction = lower (direction);
0076 
0077 if (strcmp (direction, 'first'))
0078     if (iscell (f))
0079         n = cellfun (@min, f);
0080     else
0081         n = f(1);
0082     end
0083 elseif (strcmp (direction, 'last'))
0084     if (iscell (f))
0085         n = cellfun (@max, f);
0086     else
0087         n = f(end);
0088     end
0089 else
0090     error ('index: DIRECTION must be either "first" or "last"');
0091 end
0092 
0093 end % function index
