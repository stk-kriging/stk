stk_select_optimizer

PURPOSE

STK_SELECT_OPTIMIZE selects an optimizer for stk_param_estim()

SYNOPSIS

function optim_num = stk_select_optimizer(bounds_available, display)

DESCRIPTION 

 STK_SELECT_OPTIMIZE selects an optimizer for stk_param_estim()

 FIXME: add documentation

 Returns
   1 for Octave / sqp
   2 for Matlab / fminsearch
   3 for Matlab / fmincon

CROSS-REFERENCE INFORMATION

This function calls:
  • stk_is_fmincon_available STK_IS_FMINCON_AVAILABLE returns true if fmincon is available
  • stk_is_octave_in_use STK_IS_OCTAVE_IN_USE returns true if the STK runs in Octave

0056     if stk_is_octave_in_use(),
0057         % We assume that the 'optim' package is installed, loaded, and

0061             optim_num_con = 1;
0062             optim_num_unc = 1;
0063         else
0064             disp('Function sqp not found !!!');

0066                 'is properly installed.']);
0067         end

0070         optim_num_con = 2 + stk_is_fmincon_available();
0071         % use fminsearch (Nelder-Mead) for unconstrained optimization
0072         optim_num_unc = 2;
0073         % TODO: use fminunc for unconstrained optimization in Matlab
0074         %       if the Optimization Toolbox is available (?)
0075     end
0076     mlock();
0077 end

0080 if bounds_available,
0081     optim_num = optim_num_con;
0082 else
0083     optim_num = optim_num_unc;
0084 end

0087 if display,
0088     
0089     fprintf('Constrained optimizer for stk_param_estim: ');
0090     switch optim_num_con
0091         case 1, % octave / sqp
0092             fprintf('sqp.\n');
0093         case 2, % Matlab / fminsearch
