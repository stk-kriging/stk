stk_set_root

PURPOSE

STK_SET_ROOT sets STK's root folder and add subfolder to the path

SYNOPSIS

function out = stk_set_root (root)

DESCRIPTION 

 STK_SET_ROOT sets STK's root folder and add subfolder to the path

CROSS-REFERENCE INFORMATION

This function calls:
  • stk_path STK_PATH returns the searchpath of STK.
This function is called by:
  • stk_get_root STK_GET_ROOT retrieves STK's root folder

SUBFUNCTIONS

  • function stk_rmpath (root)
  • function stk_addpath (root)
  • function root = find_stk_root ()

0030 function out = stk_set_root (root)
0031 
0032 current_root = find_stk_root ();
0033 
0034 if nargin > 0,
0035         
0036     while ~ isempty (current_root) && ~ strcmp (current_root, root)                
0037         warning (sprintf (['Removing another copy of STK from the ' ...
0038             'search path.\n    (%s)\n'], current_root)); %#ok<SPWRN>
0039         stk_rmpath (current_root);
0040         current_root = find_stk_root ();
0041     end
0042     
0043     % NOTE: calling stk_rmpath when root and stkRootFolder are identical
0044     % is harmless in recent versions of Matlab and Octave, but has been
0045     % found to cause a bug in Octave 3.0.2 when calling stk_init twice
0046     % in a row.
0047     
0048     current_root = root;
0049     
0050     % finally, add STK folders to the path
0051     stk_addpath (current_root);
0052     
0053 end
0054 
0055 out = current_root;
0056 
0057 end % function stk_set_root

0078 function stk_addpath (root)
0079 
0080 path = stk_path (root);
0081 
0082 for i = 1:length (path),
0083     if exist (path{i}, 'dir')
0084         addpath (path{i});
0085     else
0086         error (sprintf (['Directory %s does not exist.\n' ...
0087             'Is there a problem in stk_path ?'], path{i}));
0088     end
0089 end
0090 
0091 end % function stk_addpath

0100 try

0102     s = which ('stk_test');
0103     % Extract root folder
0104     n = length (s) - 1 - length (fullfile ('misc', 'test', 'stk_test.m'));
0105     root = s(1:n);
0106 catch
0107     root = [];
0108 end
