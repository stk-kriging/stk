CROSS-REFERENCE INFORMATION

This function calls:
  • stk_get_root STK_GET_ROOT retrieves STK's root folder
  • stk_is_octave_in_use STK_IS_OCTAVE_IN_USE returns true if the STK runs in Octave
This function is called by:
  • stk_set_root STK_SET_ROOT sets STK's root folder and add subfolder to the path
