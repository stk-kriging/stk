stk_materncov_iso

PURPOSE

STK_MATERNCOV_ISO computes the isotropic Matern covariance

SYNOPSIS

function k = stk_materncov_iso( param, x, y, diff )

DESCRIPTION 

 STK_MATERNCOV_ISO computes the isotropic Matern covariance

 CALL: [k]=stk_materncov_iso(param, x, y, diff)
   param  = vector of parameters of size 3
   x      = structure whose field 'a' contains the observed points.
            x.a  is a matrix of size n x d, where n is the number of
            points and d is the dimension of the
            factor space
   y      = same as x
   diff   = differentiation parameter

 STK_MATERNCOV_ISO computes a Matern covariance between two random vectors
 specified by the locations of the observations. The isotropic covariance
 function has three parameters
    param(1) = log(sigma^2) is the logarithm of the variance of random
               process
    param(2) = log(nu) is the logarithm of the regularity parameter of the
               Matern covariance
    param(3) = -log(rho) is the logarithm of the inverse of the range
               parameter
 If diff ~= -1, the function returns the derivative of the covariance wrt
 param(diff)

0055 stk_narginchk(3, 4);

0057 % default: compute the value (not a derivative)
0058 if (nargin<4), diff = -1; end
0059 
0060 % extract parameters from the "param" vector
0061 Sigma2 = exp(param(1));
0062 Nu     = exp(param(2));
0063 invRho = exp(param(3));
0064 
0065 % check parameter values
0066 if ~(Sigma2>0) || ~(Nu>0) || ~(invRho>0),
0067     error('Incorrect parameter value.');
0068 end
0069 
0070 % check if all input arguments are the same as before
0071 % (or if this is the first call to the function)
0072 if isempty(x0) || isempty(y0) || isempty(param0) || ...
0073         ~isequal({x.a,y.a,param},{x0.a,y0.a,param0})
0074     % compute the distance matrix
0075     D  = invRho * stk_distance_matrix(x.a,y.a);
0076     % save arguments for the nex call
0077     x0 = x; y0 = y; param0 = param;
0078 end
0079 
0080 
0081 if diff == -1,
0082     %%% compute the value (not a derivative)
0083     k = Sigma2 * stk_sf_matern(Nu, D, -1);
0084 elseif diff == 1,
0085     %%% diff wrt param(1) = log(Sigma2)
0086     k = Sigma2 * stk_sf_matern(Nu, D, -1);
0087 elseif diff == 2,
0088     %%% diff wrt param(2) = log(Nu)
0089     k = Nu * Sigma2 * stk_sf_matern(Nu, D, 1);
0090 elseif diff == 3,
0091     %%% diff wrt param(3) = - log(invRho)
0092     k = D .* (Sigma2 * stk_sf_matern(Nu, D, 2));
0093 else
0094     error('there must be something wrong here !');
0095 end
