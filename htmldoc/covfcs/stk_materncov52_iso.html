stk_materncov52_iso

PURPOSE

STK_MATERNCOV52_ISO computes the isotropic Matern covariance

SYNOPSIS

function k = stk_materncov52_iso(param, x, y, diff, pairwise)

DESCRIPTION 

 STK_MATERNCOV52_ISO computes the isotropic Matern covariance

 CALL: k = stk_materncov52_iso(param, x, y, diff)
   param  = vector of parameters of size 2
   x      = structure whose field 'a' contains the observed points.
            x.a  is a matrix of size n x d, where n is the number of
            points and d is the dimension of the
            factor space
   y      = same as x
   diff   = differentiation parameter

 STK_MATERNCOV52_ISO computes a Matern covariance between two random vectors
 specified by the locations of the observations. The isotropic covariance
 function has three parameters
    param(1) = log(sigma^2) is the logarithm of the variance of random
               process
    param(2) = -log(rho) is the logarithm of the inverse of the range
               parameter
 If diff ~= -1, the function returns the derivative of the covariance wrt
 param(diff)

0054 % process input arguments
0055 if isstruct(x), x = x.a; end

0060 % extract parameters from the "param" vector
0061 Sigma2 = exp(param(1));
0062 invRho = exp(param(2));
0063 
0064 % check parameter values
0065 if ~(Sigma2>0) || ~(invRho>0),
0066     error('Incorrect parameter value.');
0067 end
0068 
0069 % check if all input arguments are the same as before
0070 % (or if this is the first call to the function)
0071 if isempty(x0) || isempty(y0) || isempty(param0) || ...
0072         ~isequal({x, y, param}, {x0, y0, param0}) || ~isequal(pairwise, pairwise0)
0073     % compute the distance matrix
0074     D  = invRho * stk_dist(x, y, pairwise);
0075     % save arguments for the nex call
0076     x0 = x; y0 = y; param0 = param; pairwise0 = pairwise;
0077 end
0078 
0079 if diff == -1,
0080     %%% compute the value (not a derivative)
0081     k = Sigma2 * stk_sf_matern52(D, -1);
0082 elseif diff == 1,
0083     %%% diff wrt param(1) = log(Sigma2)
0084     k = Sigma2 * stk_sf_matern52(D, -1);
0085 elseif diff == 2,
0086     %%% diff wrt param(3) = - log(invRho)
0087     k = D .* (Sigma2 * stk_sf_matern52(D, 1));
0088 else
0089     error('there must be something wrong here !');
0090 end
