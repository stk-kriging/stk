stk_gausscov_iso

PURPOSE

STK_GAUSSCOV_ISO computes the isotropic Gaussian covariance function

SYNOPSIS

function k = stk_gausscov_iso (param, x, y, diff, pairwise)

DESCRIPTION 

 STK_GAUSSCOV_ISO computes the isotropic Gaussian covariance function

 CALL: K = stk_gausscov_iso (PARAM, X, Y)

   computes  the covariance matrix K  between  the sets of locations  X  and Y, 
   using the isotropic Gaussian covariance function with parameters PARAM.  The
   output matrix K has size NX x NY, where NX is the number of rows in X and NY
   the number of rows in Y. The vector of parameters must have two elements:

     * PARAM(1) = log (SIGMA ^ 2), where SIGMA is the standard deviation,

     * PARAM(2) = - log (RHO), where RHO is the range parameter.

 CALL: dK = stk_gausscov_iso (PARAM, X, Y, DIFF)

   computes the derivative of the covariance matrix with respect to PARAM(DIFF)
   if DIFF is equal to 1 or 2, or the covariance matrix itself if DIFF is equal
   to -1 (in which case this is equivalent to stk_gausscov_iso (PARAM, X, Y)).

 CALL: K = stk_gausscov_iso (PARAM, X, Y, DIFF, PAIRWISE)

   computes the covariance vector  (or a derivative of it if DIFF > 0)  between
   the sets of locations X and Y.  The output K is a vector of length N,  where
   N is the common number of rows of X and Y.

 See also: stk_sf_gausscorr, stk_gausscov_aniso

0055 function k = stk_gausscov_iso (param, x, y, diff, pairwise)
0056 
0057 if nargin > 5,
0058     stk_error ('Too many input arguments.', 'TooManyInputArgs');
0059 end
0060 
0061 persistent x0 y0 param0 pairwise0 D
0062 
0063 % process input arguments
0064 x = double (x);
0065 y = double (y);
0066 if nargin < 4, diff = -1; end
0067 if nargin < 5, pairwise = false; end
0068 
0069 % extract parameters from the "param" vector
0070 Sigma2 = exp (param(1));
0071 invRho = exp (param(2));
0072 
0073 % check parameter values
0074 if ~ (Sigma2 > 0) || ~ (invRho > 0),
0075     error ('Incorrect parameter value.');
0076 end
0077 
0078 % check if all input arguments are the same as before
0079 % (or if this is the first call to the function)
0080 if isempty (x0) || isempty (y0) || isempty (param0) || ...
0081         ~ isequal ({x, y, param}, {x0, y0, param0}) || ...
0082         ~ isequal (pairwise, pairwise0)
0083     % compute the distance matrix
0084     D  = invRho * stk_dist (x, y, pairwise);
0085     % save arguments for the nex call
0086     x0 = x;  y0 = y;  param0 = param;  pairwise0 = pairwise;
0087 end
0088 
0089 if diff == -1,
0090     % compute the value (not a derivative)
0091     k = Sigma2 * stk_sf_gausscorr (D, -1);
0092 elseif diff == 1,
0093     % diff wrt param(1) = log(Sigma2)
0094     k = Sigma2 * stk_sf_gausscorr (D, -1);
0095 elseif diff == 2,
0096     % diff wrt param(3) = - log(invRho)
0097     k = D .* (Sigma2 * stk_sf_gausscorr (D, 1));
0098 else
0099     error('there must be something wrong here !');
0100 end
