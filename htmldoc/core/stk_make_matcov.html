stk_make_matcov

PURPOSE

STK_MAKE_MATCOV computes a covariance matrix (and a design matrix).

SYNOPSIS

function [K, P] = stk_make_matcov(model, x0, x1)

DESCRIPTION 

 STK_MAKE_MATCOV computes a covariance matrix (and a design matrix).

 CALL: [K, P] = stk_make_matcov(MODEL, X0)

    computes the covariance matrix K and the design matrix P for the model
    MODEL at the set of points X0. For a set of N points on a DIM-dimensional
    space of factors, X0 is expected to be a structure whose field 'a' contains
    an N x DIM matrix. As a result, a matrix K of size N x N and a matrix P of
    size N x L are obtained, where L is the number of regression functions in
    the linear part of the model; e.g., L = 1 if MODEL.order is zero (ordinary
    kriging).

 CALL: K = stk_make_matcov(MODEL, X0, X1)

    computes the covariance matrix K for the model MODEL between the sets of
    points X0 and X1. Both X0 and X1 are expected to be structures with an 'a'
    field, containing the actual numerical data. The resulting K matrix is of
    size N0 x N1, where N0 is the number of rows of XO.a and N1 the number of
    rows of X1.a.

 BE CAREFUL: 
    
    stk_make_matcov(MODEL, X0) and stk_makematcov(MODEL, X0, X0) are NOT 
    equivalent if model.lognoisevariance exists (in the first case, the
    noise variance is added on the diagonal of the covariance matrix).

0060 switch nargin
0061     case 2, % stk_make_matcov(model, x0)
0062         make_matcov_auto = true;
0063     case 3, % stk_make_matcov(model, x0, x1)
0064         make_matcov_auto = false;
0065     otherwise
0066         error('Incorrect number of input arguments.');
0067 end

0069 if isfield(model, 'Kx_cache'),
0070     
0071     % handle the case where a 'Kx_cache' field is present
0072     if make_matcov_auto,
0073         K = model.Kx_cache(x0, x0);
0074     else
0075         K = model.Kx_cache(x0, x1);
0076     end
0077     
0078 else % handle the case where the covariance matrix must be computed
0079     
0080     %=== blocking parameters for parallel computing
0081     
0082     % If the size of the covariance matrix to be computed is smaller than
0083     % MIN_SIZE_FOR_BLOCKING, we don't even consider using parfor.
0084     MIN_SIZE_FOR_BLOCKING = 500^2;
0085     
0086     % If it is decided to use parfor, the number of blocks will be chosen
0087     % in such a way that blocks smaller than MIN_BLOCK_SIZE are never used
0088     MIN_BLOCK_SIZE = 100^2;
0089     
0090     %=== decide whether parallel computing should be used or not
0091     
0092     if isfield(model,'Kx_cache'), % SYNTAX: model, x(indices)
0093         ncores = 1; % avoids a call to matlabpool() which is slow
0094     else
0095         N = size(x0.a,1);
0096         if make_matcov_auto, N=N*N; else N=N*size(x1.a,1); end
0097         if (N < MIN_SIZE_FOR_BLOCKING) || ~stk_is_pct_installed(),
0098             ncores = 1; % do not use parallel computing
0099         else
0100             ncores = max( 1, matlabpool('size') );
0101             % note: matlabpool('size') returns 0 if the PCT is not started
0102         end
0103     end
0104     
0105     %=== call the subfunction that does the actual computations
0106     
0107     if make_matcov_auto,
0108         %
0109         % FIXME: avoid computing twice each off-diagonal term
0110         %
0111         if ncores == 1, % shortcut when parallelization is not used
0112             K = feval( model.covariance_type, model.param, x0, x0 );
0113         else
0114             K = stk_make_matcov_auto_parfor( model, x0, ncores, MIN_BLOCK_SIZE );
0115         end
0116         if isfield( model, 'lognoisevariance' ),
0117             K = K + stk_noisecov( size(K,1), model.lognoisevariance );
0118         end
0119     else
0120         if ncores == 1, % shortcut when parallelization is not used
0121             K = feval( model.covariance_type, model.param, x0, x1 );
0122         else
0123             K = stk_make_matcov_inter_parfor( model, x0, x1, ncores, MIN_BLOCK_SIZE );
0124         end
0125     end
0126     
0127 end
0128 
0129 %=== compute the regression functions
0130 
0131 if nargout > 1, P = stk_ortho_func(model, x0); end

0133 end
