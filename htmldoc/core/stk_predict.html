stk_predict

PURPOSE

STK_PREDICT performs a kriging prediction from data

SYNOPSIS

function [zp, lambda, mu] = stk_predict(xi, zi, xt, model, varargin)

DESCRIPTION 

 STK_PREDICT performs a kriging prediction from data 

 CALL: [zp, lambda, mu] = stk_predict(xi, zi, xt, model,...)

 STK_PREDICT computes a kriging approximation given data and a model

 FIXME: documentation incomplete

 USE #1: if model.Kx_cache exist,
   - it is assumed that xi and xt are indices (integers)
   - xi is required
   - xt is optional (can be empty)

 USE #2: otherwise,
   - xi and xt are expected to be structures, whose field 'a' contains
     the observed points (matrix of size n x d, where n is the number 
     of points and d is the dimension of the factor space)
   - both arguments are required
 
 EXAMPLE: see examples/example01.m

CROSS-REFERENCE INFORMATION

This function calls:
  • stk_make_matcov STK_MAKE_MATCOV computes a covariance matrix
  • stk_is_octave_in_use STK_IS_OCTAVE_IN_USE returns true if the STK runs in Octave
This function is called by:
  • example01 Example 01 constructs a kriging approximation in 1D
  • example02 Example 02 constructs a kriging approximation in 1D
  • example03 Example 03 constructs a kriging approximation in 2D
  • example05 Example 05 generates conditioned sample paths
