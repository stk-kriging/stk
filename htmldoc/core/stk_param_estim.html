stk_param_estim

PURPOSE

STK_PARAM_ESTIM estimates the parameters of the covariance from data

SYNOPSIS

function paramopt = stk_param_estim(param0, xi, yi, model)

DESCRIPTION

 STK_PARAM_ESTIM helper function to estimate the parameters of a
