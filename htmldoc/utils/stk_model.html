stk_model

PURPOSE

STK_MODEL generates a model with default covariance parameters.

SYNOPSIS

function model = stk_model (covariance_type, varargin)

DESCRIPTION 

 STK_MODEL generates a model with default covariance parameters.

 CALL: MODEL = stk_model()

   returns a structure MODEL (see below for a description of the fields in such
   as structure) corresponding to one-dimensional Gaussian process prior with a
   constant but unknown mean ("ordinary" kriging) and a stationary Matern
   covariance function.

 CALL: MODEL = stk_model(COVARIANCE_TYPE)

   uses the user-supplied COVARIANCE_TYPE instead of the default.

 CALL: MODEL = stk_model(COVARIANCE_TYPE, DIM)

   creates a DIM-dimensional model. Note that, for DIM > 1, anisotropic
   covariance functions are provided with default parameters that make them
   isotropic.

 In STK, a Gaussian process model is described by a 'model' structure,
 which has mandatory fields and optional fields.

   MANDATORY FIELDS: covariance_type, param
   OPTIONAL FIELD: lognoisevariance

 See also stk_materncov_iso, stk_materncov_aniso, ...

0104 function model = stk_model_ (covariance_type, dim)
0105 
0106 if nargin > 2,
0107    stk_error ('Too many input arguments.', 'TooManyInputArgs');
0108 end
0109 
0110 model = struct();
0111 
0112 %%% covariance type
0113 
0114 model.covariance_type = covariance_type;
0115 
0116 %%% model.order
0117 
0118 % use ordinary kriging as a default choice
0119 model.order = 0;
0120 
0121 %%% model.dim
0122 
0123 % default dimension is d = 1
0124 if nargin < 2,
0125     model.dim = 1;
0126 else
0127     model.dim = dim;
0128 end
0129 
0130 %%% model.param
0131 
0132 VAR0 = 1.0; % default value for the variance parameter
0133 
0134 switch model.covariance_type
0135     
0136     case 'stk_materncov_iso'
0137         
0138         NU0 = 2.0;   % smoothness (regularity) parameter
0139         RHO = 0.3;   % range parameter (spatial scale)
0140         
0141         model.param = log([VAR0; NU0; 1/RHO]);
0142         
0143     case {'stk_materncov32_iso', 'stk_materncov52_iso'}
0144         
0145         RHO = 0.3;   % range parameter (spatial scale)
0146         
0147         model.param = log([VAR0; 1/RHO]);
0148         
0149     case 'stk_materncov_aniso'
0150         
0151         NU0 = 2.0;   % smoothness (regularity) parameter
0152         RHO = 0.3;   % range parameter (spatial scale)
0153         
0154         model.param = log([VAR0; NU0; 1/RHO * ones(model.dim,1)]);
0155         
0156     case {'stk_materncov32_aniso', 'stk_materncov52_aniso'}
0157         
0158         RHO = 0.3;   % range parameter (spatial scale)
0159         
0160         model.param = log([VAR0; 1/RHO * ones(model.dim,1)]);
0161         
0162     otherwise
0163         
0164         warning('Unknown covariance type, model.param cannot be initialized.');
0165         
0166         model.param = [];
