stk_model

PURPOSE

STK_MODEL generates a model with default covariance parameters.

SYNOPSIS

function model = stk_model(covariance_type, dim)

DESCRIPTION 

 STK_MODEL generates a model with default covariance parameters.

 CALL: MODEL = stk_model()

   returns a structure MODEL (see below for a description of the fields in such
   as structure) corresponding to one-dimensional Gaussian process prior with a
   constant but unknown mean ("ordinary" kriging) and a stationary Matern
   covariance function.

 CALL: MODEL = stk_model(COVARIANCE_TYPE)

   uses the user-supplied COVARIANCE_TYPE instead of the default.

 CALL: MODEL = stk_model(COVARIANCE_TYPE, DIM)

   creates a DIM-dimensional model. Note that, for DIM > 1, anisotropic
   covariance functions are provided with default parameters that make them
   isotropic.

 In STK, a Gaussian process model is described by a 'model' structure,
 which has mandatory fields and optional fields.

   MANDATORY FIELDS: covariance_type, param
   OPTIONAL FIELDS: Kx_cache, lognoisevariance

 See also stk_materncov_iso, stk_materncov_aniso, ...

0055 function model = stk_model(covariance_type, dim)
0056 
0057 stk_narginchk(0, 2);
0058 
0059 model = struct();
0060 
0061 %%% covariance type
0062 
0063 if nargin < 1,
0064     % use the (isotropic) Matern covariance function as a default choice
0065     % (note that, since nargin == 0 here, a one-dimensional will be produced)
0066     model.covariance_type = 'stk_materncov_iso';
0067 else
0068     model.covariance_type = covariance_type;
0069 end
0070 
0071 %%% model.order
0072 
0073 % use ordinary kriging as a default choice
0074 model.order = 0;
0075 
0076 %%% model.dim
0077 
0078 % default dimension is d = 1
0079 if nargin < 2,
0080     model.dim = 1;
0081 else
0082     model.dim = dim;
0083 end
0084 
0085 %%% model.param
0086 
0087 VAR0 = 1.0; % default value for the variance parameter
0088 
0089 switch model.covariance_type
0090     
0091     case 'stk_materncov_iso'
0092         
0093         NU0 = 2.0;   % smoothness (regularity) parameter
0094         RHO = 0.3;   % range parameter (spatial scale)
0095         
0096         model.param = log([VAR0; NU0; 1/RHO]);
0097         
0098     case {'stk_materncov32_iso', 'stk_materncov52_iso'}
0099         
0100         RHO = 0.3;   % range parameter (spatial scale)
0101         
0102         model.param = log([VAR0; 1/RHO]);
0103         
0104     case 'stk_materncov_aniso'
0105         
0106         NU0 = 2.0;   % smoothness (regularity) parameter
0107         RHO = 0.3;   % range parameter (spatial scale)
0108         
0109         model.param = log([VAR0; NU0; 1/RHO * ones(model.dim,1)]);
0110         
0111     case {'stk_materncov32_aniso', 'stk_materncov52_aniso'}
0112         
0113         RHO = 0.3;   % range parameter (spatial scale)
0114         
0115         model.param = log([VAR0; 1/RHO * ones(model.dim,1)]);
0116         
0117     otherwise
0118         
0119         warning('Unknown covariance type, model.param cannot be initialized.');
0120         
0121         model.param = [];
0122         
0123 end % switch
