apply

PURPOSE

APPLY maps a function to the rows or columns of a dataframe.

SYNOPSIS

function z = apply (x, dim, F, u)

DESCRIPTION

0029 function z = apply (x, dim, F, u)
0030 
0031 if nargin == 4
0032     uu = {u};
0033 else
0034     uu = {};
0035 end
0036 
0037 if dim == 1
0038     % act along columns
0039     z = feval (F, x.data, uu{:}, 1);
0040 else
0041     if dim ~= 2,
0042         stk_error ('Incorrect dimension specifier', 'IncorrectDimSpec');
0043     else
0044         % act along rows (less usual)
0045         z = feval (F, x.data, uu{:}, 2);
0046     end
0047 end
