stk_check_octave_packages

PURPOSE

STK_CHECK_OCTAVE_PACKAGES checks required Octave packages

SYNOPSIS

function stk_check_octave_packages()

DESCRIPTION 

 STK_CHECK_OCTAVE_PACKAGES checks required Octave packages

 CALL: stk_check_octave_packages()

 An error is raised is one of the required packages is not installed.
 Otherwise, all required packages are loaded (if they are not already).

0072 function stk_check_octave_package_(name, pkg_list)
0073 
0074 for i = 1:length(pkg_list)
0075   if strcmp(pkg_list{i}.name, name),
0076     if ~pkg_list{i}.loaded,
0077       pkg('load', name);
0078     end
0079     fprintf('Octave package %s-%s loaded.\n', ...
0080         pkg_list{i}.name, pkg_list{i}.version);
0081     return
0082   end
0083 end
0084 
0085 error('Octave package %s not installed.', name);
0086 
0087 end
